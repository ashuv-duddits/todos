## Скрипты package.json:

| Скрипт | Назначение |
| ------ | ------ |
| dev | Разработка проекта в стиле **SPA** на **Vue.js** |
| build  | Сборка проекта. Все файлы будут доступны из директории **dist** |

#### Чтобы запустить скрипт:
```sh
$ npm run имя_скрипта
```
